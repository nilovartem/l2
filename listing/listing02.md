Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1
Причина такого поведения описана в документации.
Из документации:
Если внешняя функция возвращает именованную переменную, то отложенная функция имеет доступ к переменной из замыкания и может ее изменять.
```
---
```
defer используется, чтобы отложить выполнение функции или выражения до возврата из внешней функции. Под возвратом подразумевается именно конец функции, а не выражение return. Если присутствует return, то отложенная функция исполняется после него.
Если имеется несколько defer, они кладут функции в стек -> функции вызываются в обратном порядке.
```
